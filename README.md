CALCULATOR
------------

A modern and responsive calculator application built using **HTML**, **CSS**, and **JavaScript**. The project performs basic arithmetic operations through an intuitive interface and demonstrates fundamental front-end web development concepts.

Features
-----------

* Perform addition, subtraction, multiplication, and division
* Clean and responsive user interface
* Real-time input and calculation
* Built with vanilla JavaScript (no external libraries)

Technologies Used
-----------------

* HTML5
* CSS3
* JavaScript (ES6)

 Getting Started
 -----------------

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start performing calculations using the on-screen buttons.

Project Structure
-----------------

```text
├── index.html
├── style.css
└── index.js
```

Purpose
-----------

This project was created to practice front-end development skills, including UI design, DOM manipulation, event handling, and JavaScript logic.
